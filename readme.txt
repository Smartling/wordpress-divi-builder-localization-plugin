Extend Smartling Connector functionality by adding localization for Divi Builder plugin

== Description ==

[The Smartling Connector](https://wordpress.org/plugins/smartling-connector) extends the WordPress interface for seamless management of the translation process. This plugin allows to localize [Divi Builder](https://www.elegantthemes.com/plugins/divi-builder/) shortcodes.
In a nutshell it just provides additional configuration how to work with Divi Builder shortcodes:
* Shortcodes should be protected from translator
* Shortcode attributes may contain content or reference to post\category\etc. Smartling translates content and localizes references
* Rest attributes are cloned from an original post into translated

Integration Features

* Automatic change detection for content updates
* Robust custom workflow engine configurable per language
* Automatic download of completed translations to WordPress
* Translation Memory integration
* No tie-ins to translation agencies or vendors
* Reporting for translation velocity, efficiency

== Installation ==

= Minimum Requirements =
* WordPress 4.6 or higher
* [Smartling Connector](https://wordpress.org/plugins/smartling-connector) 1.4.4 or higher

1. Upload the plugin files to the `/wp-content/plugins/smartling-divi-builder-localization` directory, or install the plugin through the WordPress plugins screen directly.
1. Go to the Plugins screen and **Network Activate** the Divi Builder localization plugin.
